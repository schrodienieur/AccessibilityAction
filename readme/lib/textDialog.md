# [textDialog](/code/lib/textDialog.bsh)

Show a text input dialog overlay and handle accept/cancel events.

&nbsp;
# Dependencies
None

&nbsp;
# How it works
`textDialog` is a helper function or object from the lib folder that provides show a text input dialog overlay and handle accept/cancel events.
It exposes methods such as `accept`.

# How to use

```java
textDialog("Enter text");
```

## Return Value
Returns a scripted textDialog object.
