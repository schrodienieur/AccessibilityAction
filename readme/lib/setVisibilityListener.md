# [setVisibilityListener](/code/lib/setVisibilityListener.bsh)

Attach a visibility listener to a view and execute callbacks when visibility changes.

# How it works
`setVisibilityListener` is a helper function or object from the lib folder that provides attach a visibility listener to a view and execute callbacks when visibility changes.

# How to use

```

```
